Usage:
	*Usage for this tool is simple. Click "Browse" and open .PAL File's filename which file you want to convert.
	*After doing so, click "Convert" and choose Save filename.
	*Remember to add .tpl at the end of your palette filename, or file will be in invalid extension.
	*After saving, wait a moment until success Message Box appears.
	*After that, opened .PAL's content (colors) will appear as review in dialog box.
	***When review dialog is being closed, it doesn't prevent you to convert multiple files! :)
--------------------------------------------------------------------------------------------------------------------------------------------------------

Building:
	***In rare cases, program may not work. If so, build it yourself:
		g++ -o ChromaFastConverter.exe -g main.cpp res.o -lgdi32 -lcomctl32 -mwindows -std=c++11 -static -static-libgcc -static-libstdc++
